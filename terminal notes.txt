Terminal Commands:

cd ./  --> "current directory"
cd . .  --> tells the current directory to move up one
cd ~  --> takes you home
cd --> "change directory"   
    *make sure to write it as:   cd Module\ 3/Day\ 2/index.js
ls --> "list"
pwd  --> "present working directory"
mkdir --> "make directory" to create a new one
touch index.js --> makes a new empty file 


React App
npx create-react-app name-app  --> create new react app
cd name-app  --> change directory to that react app
npm start --> start running the react app

highlight text and press: Command d  --> selects and changes the same value throughout the same page
